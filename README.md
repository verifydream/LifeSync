## âš¡ Tentang LifeSync

**LifeSync** bukan sekadar aplikasi *to-do list*. Ini adalah sistem operasi kehidupan pribadi yang dibangun dengan desain **Glassmorphism** modern. Aplikasi ini menggabungkan pelacakan keuangan, manajemen kebiasaan (atomic habits), jurnal suasana hati, dan mode fokus (pomodoro) ke dalam satu tampilan dasbor yang intuitif.

Dibuat dengan **React 19** dan **TypeScript**, LifeSync dirancang untuk performa tinggi, interaktivitas yang mulus, dan tampilan yang memanjakan mata (mendukung Dark Mode sepenuhnya).
